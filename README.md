# CivilCalc

CivilCalc adalah aplikasi web sederhana untuk membantu mahasiswa Teknik Sipil melakukan perhitungan dasar konstruksi dengan cepat dan mudah.

## Fitur

- Menghitung luas bangunan
- Menghitung volume beton
- Mengestimasi kebutuhan semen
- Antarmuka sederhana dan mudah digunakan
- Dapat diakses melalui browser tanpa instalasi

## Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript

## Cara Menjalankan

1. Clone repository ini:

```bash
git clone https://github.com/fauzanabdullah/civilcalc.git
```

2. Masuk ke folder project:

```bash
cd civilcalc
```

3. Buka file `index.html` menggunakan browser.

## Cara Menggunakan

### Hitung Luas

Masukkan:
- Panjang (m)
- Lebar (m)

Kemudian klik tombol **Hitung Luas**.

### Hitung Volume Beton

Masukkan:
- Panjang (m)
- Lebar (m)
- Tinggi (m)

Kemudian klik tombol **Hitung Volume**.

### Hitung Kebutuhan Semen

Aplikasi menggunakan asumsi:

```text
1 m³ beton ≈ 8 sak semen
```
